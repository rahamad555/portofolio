# 📌 Deskripsi Website
Project ini merupakan website portfolio pribadi yang dikembangkan sebagai bagian dari Mini Project 2 Praktikum Pemrograman Berbasis Web. Website ini telah dikembangkan dari versi statis (HTML & CSS) menjadi website dinamis menggunakan PHP dan MySQL.

Data yang ditampilkan seperti profil, skills, experience, dan certificates diambil langsung dari database sehingga dapat dikelola dengan lebih fleksibel.

# 🎯 Fitur Utama
✅ Tampilan Home (Hero Section)
✅ About Me (Profil singkat)
✅ Skills dengan progress bar dinamis
✅ Experience (Pengalaman)
✅ Certificates (Sertifikat)
✅ Smooth scrolling navigation
✅ Data terintegrasi dengan database MySQL


# 🛠️ Teknologi yang Digunakan
HTML5
CSS
PHP (Native)
MySQL
Laragon (Local Server)
phpMyAdmin




# 🗄️ Struktur Database
Database: portfolio_db

Tabel:

profile → nama, deskripsi
skills → nama_skill, persen
experience → pengalaman
certificates → judul, deskripsi, gambar



# 🚀 Cara Menjalankan Project
Jalankan Laragon
Simpan project di folder: C:\laragon\www\portofolio
Import database ke phpMyAdmin
Pastikan koneksi di koneksi.php: mysqli_connect("localhost", "root", "", "portfolio_db");
Jalankan di browser: http://localhost/portofolio/
